# Healthy Habits アプリ（シンプル版）

管理栄養士ふみが作成した、習慣を楽しく続けるための健康管理アプリです。

## ✨ 特徴

### セキュリティ重視
- SHA-256パスワード暗号化
- ソルト付きハッシュ化
- ユーザーデータ完全分離
- 24時間セッション管理

### シンプルで安定
- 必要な機能だけに絞った設計
- 軽快な動作
- エラーが起きにくい構造

## 🎯 主な機能

### ✅ 残した機能
- ユーザー登録・ログイン（安全な認証）
- 6つの健康項目記録
  - 食事（朝・昼・夜・間食 + 自由記録）
  - 水分（±ボタンで簡単入力）
  - 運動（±10分ボタン）
  - 睡眠（±0.5時間ボタン）
  - マインド（0-10スライダー）
  - 体重（シンプルなグラフ表示）
- 目標設定機能
- 過去記録の閲覧（日付切り替え）
- メッセージ機能（ユーザー→管理栄養士）
- 励ましメッセージ

### ❌ 削除した機能（安定性のため）
- メダル機能
- 管理者ダッシュボード
- パスワードリセット
- 達成ボタン
- 写真アップロード

## 🚀 Vercelへのデプロイ

### 必要なもの
- GitHubアカウント
- Vercelアカウント（無料）

### デプロイ手順

1. **GitHubにコードをアップロード**
   - このフォルダの全ファイルをGitHubリポジトリにアップロード

2. **Vercelでプロジェクトをインポート**
   - https://vercel.com にアクセス
   - 「Add New」→「Project」をクリック
   - GitHubリポジトリを選択
   - 「Import」をクリック

3. **設定確認**
   - Framework Preset: **Next.js**（自動選択）
   - Root Directory: そのまま
   - 「Deploy」をクリック

4. **完成！**
   - 3〜5分でデプロイ完了
   - URLが表示されます

## 📱 使い方

1. アカウント登録
2. 記録したい項目を選択（設定タブ）
3. 毎日記録を入力
4. 過去の記録を振り返る

## 🔒 セキュリティについて

- パスワードは暗号化されて保存
- 個人データは完全に分離
- LocalStorageを安全に使用
