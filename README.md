# 健康トラッカーアプリ

管理栄養士が作成した、習慣を楽しく続けるための健康管理アプリです。

## 🚀 Vercelへのデプロイ方法（初心者向け）

### 必要なもの
- GitHubアカウント
- Vercelアカウント（無料）

### デプロイ手順

#### 1. GitHubにコードをアップロード

1. **GitHubにログイン**
   - https://github.com にアクセス
   - アカウントがない場合は「Sign up」から作成

2. **新しいリポジトリを作成**
   - 右上の「+」→「New repository」をクリック
   - Repository name: `health-tracker-app`（任意の名前でOK）
   - Public または Private を選択
   - 「Create repository」をクリック

3. **コードをアップロード**
   - 作成したリポジトリのページで「uploading an existing file」をクリック
   - このフォルダ内のすべてのファイルをドラッグ&ドロップ
   - 「Commit changes」をクリック

#### 2. Vercelでデプロイ

1. **Vercelにログイン**
   - https://vercel.com にアクセス
   - 「Sign Up」をクリック
   - 「Continue with GitHub」でGitHubアカウントと連携

2. **プロジェクトをインポート**
   - ダッシュボードで「Add New」→「Project」をクリック
   - 先ほど作成したGitHubリポジトリを選択
   - 「Import」をクリック

3. **設定を確認**
   - Framework Preset: Next.js（自動で選択されます）
   - Root Directory: そのまま（変更不要）
   - 「Deploy」をクリック

4. **デプロイ完了！**
   - 数分待つとデプロイが完了します
   - 表示されるURLをクリックすると、あなたのアプリが公開されています！

### 📱 アプリの使い方

- ログインページで新規登録してください
- 健康習慣を記録して、メダルを集めましょう！

### 🔄 アプリを更新したい場合

GitHubのリポジトリを更新すると、Vercelが自動的に再デプロイしてくれます。
