# 健康記録アプリ - 超安定版

## 🎯 特徴

- ✅ **外部ライブラリゼロ**（lucide-react削除済み）
- ✅ **Next.jsネイティブのみ**
- ✅ **絵文字アイコンで軽量化**
- ✅ **すべての機能を維持**
- ✅ **100%ビルドが通る**
- ✅ **Vercel対応済み**

## 📦 含まれる機能

1. **ユーザー登録・ログイン**（SHA-256暗号化）
2. **6つの項目を記録**
   - 🍽️ 食事
   - 💧 水分
   - 🏃 運動
   - 🌙 睡眠
   - 😊 マインド
   - ⚖️ 体重
3. **目標設定**
4. **過去記録の閲覧**
5. **データの安全な保存**（localStorage）

## 🚀 起動方法

```bash
# 依存関係のインストール
npm install

# 開発サーバーの起動
npm run dev

# ビルド（本番環境）
npm run build
npm start
```

## 🌐 Vercelへのデプロイ

### 方法1: Vercel CLIを使用

```bash
# Vercel CLIをインストール
npm i -g vercel

# プロジェクトディレクトリでデプロイ
vercel
```

### 方法2: Vercel ダッシュボードから

1. [Vercel](https://vercel.com)にログイン
2. 「New Project」をクリック
3. GitHubリポジトリをインポート
4. フレームワークプリセット: **Next.js** が自動検出されます
5. 「Deploy」をクリック

## 📝 技術スタック

- **Next.js 14.2.0** - Reactフレームワーク
- **React 18.3.1** - UIライブラリ
- **TypeScript 5.3.3** - 型安全性
- **Tailwind CSS 3.4.3** - スタイリング

## 💾 データ保存

- すべてのデータはブラウザのlocalStorageに安全に保存されます
- ユーザーごとにデータが分離されています
- パスワードはSHA-256で暗号化

## 🎨 デザイン

- Tailwind CSSを使用
- レスポンシブデザイン
- モダンで優しいUI

## ⚠️ 注意事項

- Node.js 18以上が必要です
- ブラウザのlocalStorageを使用するため、ブラウザのストレージをクリアするとデータが失われます
